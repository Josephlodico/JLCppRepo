## Assignment Details

### First C++ Exercise — `FirstExercise.cpp`
A console-based user account system demonstrating:
- **Classes & Objects** — `UserAccount` class with private member variables
- **Constructors** — initializer list syntax
- **Member Functions** — `Login()` method with credential validation
- **Getters** — `getID()` accessor method
- **Control Flow** — `while` loop for repeated login attempts
- **Input Handling** — `cin`, `getline`, and `cin.ignore()` usage

### Pointers Assignment — `Pointers Assignment.cpp`
A series of exercises covering basic pointer operations, including:
- Declaring and initializing pointers
- Dereferencing to modify values
- Analysis of output, compilation behavior, and memory safety

## Setup & Build

**Requirements:** Visual Studio 2022 (v143 toolset), Windows 10 SDK

1. Clone the repository
2. Open `JLCppRepo.sln` in Visual Studio
3. Select a build configuration (`Debug|x64` recommended)
4. Press **Ctrl+F5** to build and run without debugging
